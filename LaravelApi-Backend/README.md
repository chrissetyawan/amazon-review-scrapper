# Laravel Api 7

Simple example of a REST API with Laravel 7.x and API for amazon review scrapper

## Install with Composer

```
    $ composer install
```

## Set Environment

```
    $ cp .env.example .env
```

## Set the application key

```
   $ php artisan key:generate
```

## Run migrations and seeds

```
   $ php artisan migrate --seed
```

## Run Application

```
   $ php artisan serve
```
